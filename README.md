# onlydole.github.io

Personal site for Taylor Dolezal.

## Stack

Static HTML and CSS.

- **Fonts**: Instrument Serif + Outfit via Google Fonts
- **Hosting**: GitHub Pages
- **Deployment**: Push to `main`

## Local Development

Open `index.html` in a browser. That's it.
